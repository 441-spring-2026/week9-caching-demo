# Week 9 Caching Demo

This simple Express app is where we will practice server- and client-side caching.

## Development

This is an [Express application](https://expressjs.com/en/5x/api/) using (Pug templating engine](https://pugjs.org/api/reference.html). It uses [nodemon](https://github.com/remy/nodemon) for hot reload on file change, so no need to restart the server after changes. [Node-cache](https://www.npmjs.com/package/node-cache) package is used for server-side caching.

On clone, run `npm install`.

Start the server with `npm run start`.
